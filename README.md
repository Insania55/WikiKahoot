### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/jorgepexp/WikiKahoot.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
 ```sh
 npm run serve
 ```
